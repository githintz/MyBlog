Private author area. Sign in with a GitHub access token that has write access to your blog repository.

How do I get a token?
  1. Open GitHub → Fine-grained tokens.
  2. Set Repository access → Only select repositories and choose your blog repo.
  3. Under Permissions → Repository → Contents, set Read and write.
  4. Generate, copy the token, and paste it above. It is stored only in this browser.
